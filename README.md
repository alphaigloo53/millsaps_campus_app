# millsaps_campus_app
The campus life app for Millsaps College where you can find info on weather and events

Description and Purpose
The Millsaps Campus Life App is my web application designed to make everyday campus life easier. Students can use this one app to check local weather and browse campus related as well as broad college news. This project was built as a final project for a web development course and brings together HTML, CSS, JavaScript, Bootstrap, and API data.

Technologies Used
HTML
CSS
JavaScript
Bootstrap 
OpenWeatherMap API 
NewsAPI 
GitHub Pages 

Setup and Deployment Instructions
Clone r download this repository to your computer
Open the project folder in VS Code
Open index.html in your browser to view the app

Features and How to Use Them
Home Page (index.html)
The landing page for the app. It has a welcome banner with a campus photo and two feature cards that link to the other pages. Use the navigation bar at the top to move between pages.
Campus News (events.html)
Type a topic into the search bar and click Search. The app will fetch real news articles related to your search from NewsAPI and display them as cards. Each card shows the article title, a short description, and a link to read the full article.
Weather (weather.html)
Click the Get Current Weather button to load the current weather for Jackson, MS. The app displays the temperature, humidity, and wind speed. A campus tip at the bottom gives a suggestion based on the current condition.
